Project Link

Automated Preventive Maintenance

"Kimberly-Clark Asia Pacific Pte Ltd"
June 2023

Spearheaded a key operational digitalization initiative by developing an Automated Preventive Maintenance (PM) System using VBA, significantly enhancing efficiency over manual planning. This system features a dynamic, intuitive user interface that provides real-time visual diagnostics of overall machine health (e.g., 99%) and section-specific conditions, using a red-light/green-light display to flag critical priority maintenance items. The solution automates the maintenance workflow, integrating a database auto-update function from user inputs and facilitating proactive maintenance scheduling via auto daily email notifications for urgent items and parts replacements. This project successfully created a comprehensive, self-managing platform for efficient and transparent maintenance planning.

Data Analyzer

"Kimberly-Clark Asia Pacific Pte Ltd"
July 2021

Developed and deployed a VBA-based "Data Analyzer" program utilized across all production machines to automate the extraction, summarization, and tabulation of daily process logs into comprehensive weekly and monthly reports. This eliminated tedious manual data consolidation, providing asset leaders with accurate, high-level analysis of the top and most frequent causes of machine delays, enabling a data-driven focus on root cause resolution.
